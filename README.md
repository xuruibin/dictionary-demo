## dictionary-demo

#### Description
~~~
This project provides methods, that can define the dictionary and break word, etc.

Default system dictionary:  "i", "like", "sam", "sung", "samsung", "mobile", "ice", "cream", "man go"
Default custom dictionary:  "i", "like", "sam", "sung", "mobile", "icecream", "man go", "mango"
~~~

#### Build
~~~
Package commands:   mvn clean install
Start commands:     java -jar web.jar
~~~

#### File
~~~
Swagger file:       http://localhost:8080/dictionary-demo/swagger-ui.html
Coverage report:    submodule/target/site/jacoco/index.html
~~~
